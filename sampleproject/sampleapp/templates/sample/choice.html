    Affichage manuel des boutons

    Il est bien sur possible de les placer à son gré, en utilisant alors le tag image_button dans son template. Ci-dessous, les boutons sont placés explicitement:
    • cliquer sur {% image_button "help" %} vous conduira à un panneau d'aide,
    • cliquer sur {% image_button "edit" %} vous conduira à une application permettant d'éditer l'objet workflow,
    • cliquer sur {% image_button "exit" %} terminera le process en vous conduisant sur l'activité finale du process
