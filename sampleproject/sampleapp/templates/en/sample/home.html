    tutorial

    Ce projet permet d'expérimenter une véritable application de workflow utilisant le moteur GoFlow. Un process workflow est ici utilisé pour permettre à l'utilisateur d'explorer les diverses application prédéfinies.

    Démarrer le tutoriel en entrant directement dans le workflow Sample process.

    modèle de projet workflow

    Le démarrage d'un projet django intégrant le moteur GoFlow comporte des tâches un peu fastidieuses (configuration settings, authentification, fichiers statiques, ...).

    Ce projet fournit une configuration standard permettant de tester immédiatement ses propres workflows, en fournissant un workflow composé d'applications types prédéfinies, qu'on peut modifier à sa guise. Un partie du tutoriel est d'ailleurs consacré à celà.
