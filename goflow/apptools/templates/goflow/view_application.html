Model view

This is a default template used by the generic application view_application.

You may define a suitable template (the default is view_application.html) with the template parameter in the urls.py file as follows: 

    (r'^my/appli/view_appli/$',
      'goflow.workflow.applications.view_application',
      {'template':'mytemplate.html'}),
This template can access the following variables:
  • object: object (linked to the process instance, instance.wfobject())
  • submit_name: submit buttons name
  • ok_values: list of submit values (button labels) for OK
  • cancel_value: submit value for cancel button

This application should be used on two cases:

  • simple view: an information panel is diplayed for user notification
  • complex edition: from this panel, you may edit a complex object; indeed the workflow object may be a complex composition of children objects. In order to complete the activity an OK button has to be clicked.
